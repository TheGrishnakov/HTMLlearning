List of useful commands

COMMAND + Z ==== UNDO

COMMAND + SHIFT + Z ==== REDO (not sure)

COMMAND + C ==== COPY

COMMAND + V ==== PASTE

HOLD OPTION + CLICK ==== MULIPLE CURSORS

COMMAND + I ==== SELECT WHOLE LINE

OPTION + LEFT/RIGHT KEYS ==== NAVIGATE

COMMAND + LEFT/RIGHT KEYS ==== START/END OF LINE

COMMAND + UP/DOWN ==== START/END OF DOCUMENT

SHIFT + OPITON + DOWN ==== COPY LINE

OPTION + UP/DOWN ==== MOVE LINE UP/DOWN

COMMAND + / = TOGGLE LINE COMMENT

COMMAND + K + COMMAND + S ==== SHORTCUTS

COMMAND + ENTER ==== START A NEW LINE
